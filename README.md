## On Building with AI

Most developers treat AI like a faster autocomplete. I treat it like a collaborator тАФ one that's available at 2am, never gets tired, and can hold an entire codebase in context.

The unlock isn't the tools. It's knowing how to:
- decompose a problem before you prompt
- iterate on outputs without losing direction  
- catch hallucinations before they become bugs
- use the right model for the right task

Still learning. Always shipping.
